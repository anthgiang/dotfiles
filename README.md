# Dotfiles

For: tmux, vim, zsh.

#### Requirements:
- zsh shell (macOS).
- zsh-autosuggestions: https://github.com/zsh-users/zsh-autosuggestions

#### Usage:
1. Clone this repository on your machine. 
```
 git clone https://github.com/anthgiang/dotfiles.git ~/.dots
```
2. Execute `dots_setup.sh`.
```
~/.dots/dots_setup.sh
```
